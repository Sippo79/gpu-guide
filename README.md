# GPU GUIDE

GPU性能比較・グラフィックボード比較サイトです。

RTX / Radeon のGPU性能を初心者向けに分かりやすく比較し、
用途別・解像度別におすすめGPUを確認できます。

---

## 主な機能

* GPU性能比較
* NVIDIA / AMD 絞り込み
* FHD / WQHD / 4K 向け分類
* GPU検索機能
* VRAM / 価格 / 消費電力表示
* GPU詳細ページ
* おすすめゲーム表示
* おすすめCPU表示
* 関連GPUリンク
